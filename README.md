# Gibwork Website

A Next.js application built with TypeScript, React, and Tailwind CSS.

## Getting Started

First, install the dependencies using one of the following commands:

```bash
# Using npm
npm install

# Using yarn
yarn install
```

### Running the Development Server

To start the development server locally:

```bash
# Using npm
npm run dev

# Using yarn
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Running Linting

To run the linter and check for code quality and style issues:

```bash
# Using npm
npm run lint

# Using yarn
yarn lint
```

### Running Tests

To execute the test suite:

```bash
# Using npm
npm run test

# Using yarn
yarn test
```

## Production Build

To build the application for production deployment:

```bash
# Using npm
npm run build
npm run start

# Using yarn
yarn build
yarn start
```
